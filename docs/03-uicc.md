# Universal Integrated Circuit Card (UICC)

## Definition

The UICC is the physical smart card inserted into a mobile device. It
hosts one or more applications such as USIM and ISIM.

## Functions

-   Hosts telecom applications
-   Secure storage
-   Application management
-   Standard communication interface

## Standards

-   ETSI TS 102 221
-   3GPP TS 31.101
